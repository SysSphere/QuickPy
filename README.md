# QuickPy 🚀

**The ultimate browser-based Python playground.**  
Copy, paste, and execute scripts instantly while mastering the language through interactive lessons and quizzes.

---

## 🌟 Key Features

*   **Instant Execution:** Run Python code directly in your browser—no installation, no servers, no fuss.
*   **Interactive Lessons:** Step-by-step tutorials that take you from `print("Hello World")` to advanced data structures.
*   **Built-in Quizzes:** Dynamic knowledge checks to ensure you're actually absorbing the concepts.
*   **Zero Latency:** Powered by **Pyodide** (WebAssembly), meaning the Python engine runs locally on your machine.
*   **Snippet Library:** Save your favorite code blocks for quick access later.
